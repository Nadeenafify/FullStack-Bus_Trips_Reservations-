# 🚌 Bus Reservation Fullstack Web App

A full-stack bus booking platform built with **Next.js, Node.js, Express, MySQL, and Stripe**, supporting trip search, seat booking, authentication, payments, and multi-language support (i18n).

## ⚙️ Features

### 🔎 Trip Search
- Search trips by **departure city, arrival city, and date**

### 🎯 Filtering System
- Filter by:
  - Price
  - Time
  - Amenities

### 🪑 Seat Selection
- Interactive seat map
- Prevent double booking
- Store selected trips in session storage

### 👤 Authentication (JWT)
- Register / Login system
- Protected routes
- Secure token-based authentication

### 💳 Payment Integration
- Stripe payment gateway
- Secure checkout flow
- Booking confirmation after payment

### 🌍 Internationalization (i18n)
- English / Arabic support
- Dynamic locale routing (`/en`, `/ar`)

### 📨 Contact System
- Send messages about the website
- Stored in database

- Node.js
